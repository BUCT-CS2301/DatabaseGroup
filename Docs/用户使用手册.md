## 1. 系统概述

### 1.1 系统简介
后台管理子系统是海外藏中国文物知识管理与服务平台的核心管理工具，为管理员提供统一的管理界面，实现对平台用户、内容、数据的全面管控。

### 1.2 主要功能模块

| 模块 | 功能描述 |
|-----|---------|
| 用户管理 | 管理系统用户，包括创建、编辑、删除、状态管理 |
| 角色权限 | 管理角色和权限，支持自定义角色 |
| 内容审核 | 审核用户生成内容，管理审核规则 |
| 数据管理 | 管理文物和博物馆数据 |
| 备份恢复 | 系统数据备份与恢复 |
| 日志管理 | 查看系统日志和操作记录 |
| 系统监控 | 查看系统统计和实时监控 |
| 系统配置 | 管理敏感词和系统公告 |

### 1.3 访问方式

- **开发环境**：http://localhost:8080
- **生产环境**：根据实际部署地址

## 2. 系统登录

### 2.1 登录页面
打开浏览器访问系统地址，进入登录页面：

![登录页面示意图]

### 2.2 登录操作
1. 在用户名输入框中输入管理员用户名
2. 在密码输入框中输入密码
3. 点击【登录】按钮

### 2.3 注意事项
- 用户名和密码区分大小写
- 连续5次登录失败，账号将被锁定30分钟
- 登录成功后系统会自动保持登录状态2小时

## 3. 用户管理

### 3.1 用户列表

#### 功能说明
查看系统所有用户列表，支持分页、搜索和筛选。

#### 操作步骤
1. 点击左侧菜单【用户管理】
2. 在搜索框中输入关键词（支持用户名、昵称、邮箱搜索）
3. 使用筛选器筛选用户状态或角色
4. 点击分页控件切换页码

#### 列表字段说明

| 字段 | 说明 |
|-----|-----|
| 用户ID | 用户唯一标识 |
| 用户名 | 登录用户名 |
| 昵称 | 用户昵称/真实姓名 |
| 邮箱 | 用户邮箱地址 |
| 状态 | ENABLED（启用）/ DISABLED（禁用） |
| 角色 | 用户所属角色 |
| 最后登录 | 最后登录时间 |

### 3.2 创建用户

#### 操作步骤
1. 在用户列表页面点击【新建用户】按钮
2. 填写用户信息：
   - **用户名**：必填，唯一标识
   - **密码**：必填，建议复杂密码
   - **昵称**：可选，显示名称
   - **邮箱**：可选，用于接收通知
   - **手机号**：可选
   - **角色**：选择用户所属角色
3. 点击【确认创建】

### 3.3 编辑用户

#### 操作步骤
1. 在用户列表中找到要编辑的用户
2. 点击【编辑】按钮
3. 修改用户信息（密码除外）
4. 点击【保存】

### 3.4 重置密码

#### 操作步骤
1. 在用户列表中找到目标用户
2. 点击【重置密码】按钮
3. 设置新密码
4. 点击【确认重置】

### 3.5 启用/禁用用户

#### 操作步骤
1. 在用户列表中找到目标用户
2. 点击【启用】或【禁用】按钮
3. 确认操作

## 4. 角色权限管理

### 4.1 角色列表

#### 功能说明
查看系统所有角色，支持搜索。

#### 操作步骤
1. 点击左侧菜单【角色管理】
2. 在搜索框中输入角色名称或代码

#### 列表字段说明

| 字段 | 说明 |
|-----|-----|
| 角色名称 | 显示名称 |
| 角色代码 | 唯一标识代码 |
| 描述 | 角色说明 |
| 是否系统角色 | 系统内置角色不可删除 |

### 4.2 创建角色

#### 操作步骤
1. 点击【新建角色】按钮
2. 填写角色信息：
   - **角色名称**：必填
   - **角色代码**：必填，唯一
   - **描述**：可选
3. 选择该角色的权限
4. 点击【确认创建】

### 4.3 编辑角色

#### 操作步骤
1. 在角色列表中找到目标角色
2. 点击【编辑】按钮
3. 修改角色信息和权限
4. 点击【保存】

### 4.4 删除角色

#### 操作步骤
1. 在角色列表中找到目标角色（非系统角色）
2. 点击【删除】按钮
3. 确认删除

### 4.5 权限管理

#### 功能说明
为角色分配权限，支持树形权限结构。

#### 操作步骤
1. 在角色列表中找到目标角色
2. 点击【权限配置】按钮
3. 在权限树中勾选需要的权限
4. 点击【保存】

## 5. 内容审核

### 5.1 审核队列

#### 功能说明
查看待审核内容列表，支持按类型和状态筛选。

#### 操作步骤
1. 点击左侧菜单【内容审核】
2. 使用筛选器选择内容类型和审核状态
3. 查看待审核内容列表

#### 列表字段说明

| 字段 | 说明 |
|-----|-----|
| 内容ID | 内容唯一标识 |
| 类型 | COMMENT/IMAGE/AUDIO/VIDEO/DYNAMIC |
| 内容摘要 | 内容预览 |
| 作者 | 内容发布者 |
| 提交时间 | 提交审核时间 |
| 状态 | PENDING/APPROVED/REJECTED |

### 5.2 审核操作

#### 审核通过
1. 在审核队列中找到目标内容
2. 点击【审核通过】按钮
3. 可选填写审核备注
4. 点击【确认】

#### 审核拒绝
1. 在审核队列中找到目标内容
2. 点击【拒绝】按钮
3. 选择拒绝原因
4. 可选填写备注
5. 点击【确认】

#### 批量审核
1. 在审核队列中勾选多条内容
2. 点击【批量通过】或【批量拒绝】
3. 确认操作

### 5.3 审核规则配置

#### 操作步骤
1. 点击左侧菜单【审核规则】
2. 设置自动审核参数：
   - 文本自动审核开关
   - 图片自动审核开关
   - 敏感词过滤开关
   - 自动审核动作（通过/拒绝/人工审核）
3. 点击【保存配置】

### 5.4 敏感词管理

#### 查看敏感词
1. 点击左侧菜单【敏感词库】
2. 查看敏感词列表

#### 添加敏感词
1. 点击【添加敏感词】按钮
2. 输入敏感词内容
3. 点击【确认添加】

#### 删除敏感词
1. 在敏感词列表中找到目标词
2. 点击【删除】按钮
3. 确认删除

## 6. 数据管理

### 6.1 文物管理

#### 文物列表
1. 点击左侧菜单【文物管理】
2. 使用搜索框搜索文物名称或编号
3. 使用筛选器筛选博物馆、类型、年代

#### 新增文物
1. 点击【新增文物】按钮
2. 填写文物信息：
   - **标题**：必填
   - **年代**：可选
   - **类型**：可选
   - **材质**：可选
   - **描述**：可选
   - **尺寸**：可选
   - **所属博物馆**：必填
   - **详情URL**：必填
   - **馆藏编号**：可选
   - **爬取日期**：必填
3. 点击【保存】

#### 编辑文物
1. 在文物列表中找到目标文物
2. 点击【编辑】按钮
3. 修改文物信息
4. 点击【保存】

#### 删除文物
1. 在文物列表中找到目标文物
2. 点击【删除】按钮
3. 确认删除（软删除，可恢复）

#### 上传文物图片
1. 在文物列表中找到目标文物
2. 点击【上传图片】按钮
3. 选择图片文件（支持JPG/PNG/GIF/WebP）
4. 点击【确认上传】

#### CSV批量导入
1. 点击【导入CSV】按钮
2. 选择CSV文件
3. 点击【开始导入】
4. 查看导入结果

#### CSV导出
1. 设置筛选条件（可选）
2. 点击【导出CSV】按钮
3. 保存导出文件

### 6.2 博物馆管理

#### 博物馆列表
1. 点击左侧菜单【博物馆管理】
2. 使用搜索框搜索博物馆名称

#### 新增博物馆
1. 点击【新增博物馆】按钮
2. 填写博物馆信息：
   - **英文名称**：必填
   - **中文名称**：可选
   - **位置**：可选
   - **官网**：可选
3. 点击【保存】

#### 编辑博物馆
1. 在博物馆列表中找到目标博物馆
2. 点击【编辑】按钮
3. 修改信息
4. 点击【保存】

#### 删除博物馆
1. 在博物馆列表中找到目标博物馆
2. 点击【删除】按钮
3. 确认删除

## 7. 备份与恢复

### 7.1 手动备份

#### 操作步骤
1. 点击左侧菜单【备份管理】
2. 点击【手动备份】按钮
3. 选择备份类型（全量/增量）
4. 填写备份描述（可选）
5. 点击【确认备份】

### 7.2 定时备份任务

#### 创建任务
1. 点击【定时任务】标签
2. 点击【新建任务】按钮
3. 填写任务信息：
   - **任务名称**：必填
   - **Cron表达式**：必填（如 "0 0 2 * * ?" 表示每日凌晨2点）
   - **备份类型**：全量/增量
   - **描述**：可选
4. 点击【保存】

#### 编辑任务
1. 在定时任务列表中找到目标任务
2. 点击【编辑】按钮
3. 修改任务信息
4. 点击【保存】

#### 删除任务
1. 在定时任务列表中找到目标任务
2. 点击【删除】按钮
3. 确认删除

#### 启用/禁用任务
1. 在定时任务列表中找到目标任务
2. 点击【启用】或【禁用】按钮

### 7.3 备份记录

#### 查看记录
1. 点击【备份记录】标签
2. 使用筛选器筛选状态和类型
3. 查看备份记录列表

#### 下载备份
1. 在备份记录列表中找到目标备份
2. 点击【下载】按钮
3. 保存备份文件

#### 删除备份
1. 在备份记录列表中找到目标备份
2. 点击【删除】按钮
3. 确认删除

### 7.4 数据恢复

#### 操作步骤
1. 在备份记录列表中找到要恢复的备份
2. 点击【恢复】按钮
3. 确认恢复操作
4. 等待恢复完成

## 11. 常见问题

### 11.1 登录问题

**问题**：登录失败，提示账号被锁定
**解决方案**：等待30分钟后重试，或联系超级管理员解锁

**问题**：忘记密码
**解决方案**：联系超级管理员重置密码

### 11.2 权限问题

**问题**：无法访问某些页面
**解决方案**：联系管理员检查角色权限配置

**问题**：操作被拒绝
**解决方案**：检查当前用户角色是否具有该操作权限

### 11.3 数据问题

**问题**：文物图片无法显示
**解决方案**：检查图片是否上传成功，或联系管理员检查存储配置

**问题**：CSV导入失败
**解决方案**：检查CSV文件格式是否正确，确保表头完整

### 11.4 备份问题

**问题**：备份失败
**解决方案**：检查存储空间是否充足，或联系管理员检查备份配置
