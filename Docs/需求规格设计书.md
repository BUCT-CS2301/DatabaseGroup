## 1. 引言

### 1.1 编写目的
本文档详细描述后台管理子系统的功能需求、非功能需求、业务流程及接口规范，作为系统设计、开发、测试和验收的依据。

### 1.2 项目背景
海外藏中国文物知识管理与服务平台是一个集文物展示、知识图谱、用户互动于一体的综合性文化平台。后台管理子系统作为平台的核心支撑，负责用户管理、内容审核、数据维护、系统监控等关键功能。

### 1.3 术语定义

| 术语 | 说明 |
|-----|-----|
| UGC | User Generated Content，用户生成内容 |
| RBAC | Role-Based Access Control，基于角色的访问控制 |
| JWT | JSON Web Token，用于身份认证 |
| Neo4j | 图数据库，用于存储知识图谱 |

---

## 2. 总体需求

### 2.1 系统定位
后台管理子系统是平台运营管理的核心工具，为管理员提供统一的管理界面，实现对平台用户、内容、数据的全面管控。

### 2.2 用户角色

#### 2.2.1 系统内置角色

| 角色 | 职责描述 |
|-----|---------|
| 超级管理员 | 拥有所有权限，可管理系统配置、分配角色权限 |
| 内容审核员 | 负责审核用户生成内容，管理审核规则 |
| 数据管理员 | 负责文物数据、博物馆数据的维护与管理 |

#### 2.2.2 自定义角色
系统支持管理员根据业务需要创建自定义角色，可灵活配置角色权限组合，满足不同管理岗位的权限需求。

## 3. 功能需求

### 3.1 用户认证与授权模块

#### 3.1.1 登录
- 管理员使用用户名/密码登录系统
- 支持JWT Token认证，Token有效期2小时
- 支持刷新Token，无需重新登录
- 记录登录日志（IP、时间、设备信息）

#### 3.1.2 权限管理
- 采用RBAC权限模型
- 支持角色的增删改查
- 支持为角色分配权限（树形权限结构）
- 支持为用户分配角色
- 系统内置角色不可删除

#### 3.1.3 用户管理
- 支持后台用户的增删改查
- 支持用户状态管理（启用/禁用）
- 支持用户角色分配
- 支持查看用户行为日志
- 支持用户违规处理（警告、禁言等）

### 3.2 内容审核模块

#### 3.2.1 审核规则配置
- 支持文本自动审核规则配置
- 支持图片自动审核规则配置
- 支持敏感词库管理（增删改查）
- 支持自动审核动作配置（通过、拒绝、人工审核）

#### 3.2.2 审核队列管理
- 支持按内容类型筛选（评论、图片、音频、视频、动态）
- 支持按审核状态筛选（待审核、已通过、已拒绝）
- 支持分页查询
- 支持查看审核详情

#### 3.2.3 审核操作
- 支持人工审核通过/拒绝
- 支持批量审核操作
- 支持填写审核备注
- 支持记录审核日志

#### 3.2.4 审核统计
- 统计总提交量、自动通过量、自动拒绝量、人工处理量
- 支持按时间范围查询

### 3.3 数据管理模块

#### 3.3.1 文物数据管理
- 支持文物数据的增删改查
- 支持软删除（数据可追溯）
- 支持按博物馆、类型、年代筛选
- 支持关键词搜索
- 支持文物图片上传（支持JPG、PNG、GIF、WebP格式）
- 支持单张图片最大10MB
- 支持CSV批量导入文物数据
- 支持CSV数据格式校验
- 支持批量导入失败回滚

#### 3.3.2 博物馆数据管理
- 支持博物馆数据的增删改查
- 支持博物馆与文物关联管理

#### 3.3.3 知识图谱数据管理
- 支持知识图谱节点和关系的查看
- 支持知识图谱数据编辑

#### 3.3.4 UGC内容管理
- 支持查看用户生成内容
- 支持删除违规内容（不可修改）

### 3.4 备份与恢复模块

#### 3.4.1 备份操作
- 支持手动触发全量/增量备份
- 支持备份文件下载
- 支持备份记录查询

#### 3.4.2 定时备份任务
- 支持创建定时备份任务
- 支持Cron表达式配置
- 支持任务启用/禁用
- 支持查看任务执行历史

#### 3.4.3 数据恢复
- 支持从备份点恢复数据
- 支持恢复任务状态跟踪

#### 3.4.4 存储管理
- 支持查看备份存储空间使用情况

### 3.5 日志管理模块

#### 3.5.1 操作日志
- 记录管理员所有操作
- 支持按用户、模块、时间范围筛选
- 支持操作详情查看

#### 3.5.2 系统日志
- 记录系统运行日志
- 支持按日志级别（DEBUG、INFO、WARN、ERROR）筛选
- 支持异常堆栈查看

#### 3.5.3 安全日志
- 记录安全事件（登录成功/失败、权限变更等）
- 支持安全事件查询

#### 3.5.4 日志导出
- 支持日志CSV格式导出
- 支持导出文件过期时间设置

### 3.6 系统监控看板

#### 3.6.1 实时概览
- 在线用户数
- 今日访问量
- 今日新增用户
- 待审核数量
- 系统CPU使用率
- JVM内存使用情况

#### 3.6.2 趋势统计
- 访问量趋势（日/周/月）
- 数据增长统计

#### 3.6.3 异常告警
- 系统异常告警展示（选做）

### 3.7 系统配置管理

#### 3.7.1 敏感词管理
- 支持敏感词增删改查

#### 3.7.2 系统公告
- 支持公告发布、编辑、删除
- 支持公告状态管理（启用/禁用）

#### 3.7.3 功能开关
- 支持功能开关配置（评论功能、AI审核、注册功能等）

## 4. 非功能需求

### 4.1 性能需求
- 系统响应时间：页面加载 < 2秒，API响应 < 500ms
- 支持并发用户数：至少100个同时在线管理员
- 数据库查询：单表查询 < 100ms，分页查询 < 500ms

### 4.2 安全需求
- 密码使用BCrypt加密存储
- 所有API需JWT认证（登录接口除外）
- 敏感操作需记录操作日志
- SQL注入防护
- XSS攻击防护
- CSRF防护

### 4.3 可用性需求
- 系统可用性：99.5%以上
- 数据备份：每日自动备份
- 故障恢复时间：< 1小时

### 4.4 可维护性需求
- 代码结构清晰，模块化设计
- 完善的日志记录
- 详细的接口文档
- 数据库表设计规范

### 4.5 兼容性需求
- 浏览器兼容性：Chrome、Firefox、Safari、Edge最新版本
- 后端支持：Java 17+
- 数据库：MySQL 8.0+
